## P4 — DECAIMENTO MENSAL POR INATIVIDADE

**Verificar primeiro:** Procurar no código e no banco por "decay", "monthly_decay", "decaimento". Verificar se existe function/trigger no Supabase.

**Se não existir, criar:**
```sql
CREATE OR REPLACE FUNCTION apply_monthly_decay()
RETURNS void AS $$
DECLARE
  decay_rate DECIMAL;
BEGIN
  SELECT CAST(value AS DECIMAL) INTO decay_rate
  FROM scoring_rules WHERE key = 'monthly_decay';
  IF decay_rate IS NULL THEN decay_rate := 0.10; END IF;

  UPDATE students
  SET total_points = GREATEST(0, FLOOR(total_points * (1 - decay_rate))),
      available_points = GREATEST(0, FLOOR(available_points * (1 - decay_rate)))
  WHERE last_action_date < NOW() - INTERVAL '30 days'
    AND total_points > 0;
END;
$$ LANGUAGE plpgsql;
```
