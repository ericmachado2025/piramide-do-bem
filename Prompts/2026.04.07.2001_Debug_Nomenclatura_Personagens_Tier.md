# NOMENCLATURA DE PERSONAGENS POR TIER
## Instrução isolada para incluir na lista de debugs

---

## PROBLEMA
Ao exibir personagens, o nome aparece isolado (ex: "Freddie Mercury") sem indicação do tier/level. Quando o mesmo personagem existe em múltiplos tiers (Fan no Tier 1 e Icon no Tier 5), o usuário não entende a diferença.

## SOLUÇÃO ESTRUTURADA

### 1. Adicionar coluna `display_prefix` na tabela `community_levels`:

```sql
ALTER TABLE community_levels ADD COLUMN IF NOT EXISTS display_prefix TEXT;
ALTER TABLE community_levels ADD COLUMN IF NOT EXISTS prefix_connector TEXT DEFAULT ' ';
```

### 2. Popular com base nos nomes existentes:

```sql
-- Atualizar display_prefix com o nome do level
UPDATE community_levels SET display_prefix = name;

-- Definir o conector por padrão
-- Nomes que funcionam como TÍTULO (sem preposição): Delta, Gamma, Beta, Alpha, Omega, Grand Master, etc.
UPDATE community_levels SET prefix_connector = ' ' 
WHERE name IN ('Delta', 'Gamma', 'Beta', 'Alpha', 'Omega', 
  'Grand Master', 'Master', 'Auror', 'Knight', 'Hokage', 'Kage',
  'Genin', 'Chunin', 'Jonin', 'Padawan', 'Youngling',
  'Epic', 'Legendary', 'Legend', 'Icon', 'Superstar', 'Platinum',
  'Gold', 'Silver', 'Bronze', 'Champion', 'Hero', 'Expert');

-- Nomes que funcionam como RELAÇÃO (com "de"): Fan, Enthusiast, Aficionado, Viewer, etc.
UPDATE community_levels SET prefix_connector = ' de ' 
WHERE name IN ('Fan', 'Enthusiast', 'Aficionado', 'Viewer', 'Superfan',
  'Beginner', 'Commoner', 'Adventurer', '1 Star', '2 Stars', '3 Stars', '4 Stars',
  'First Year', 'OWL Student', 'NEWT Student', 'Estudante OWL', 'Estudante NEWT', 'Primeiro Ano');

-- Nomes numéricos (Level 1-4, Nível 1-4): usar "de"  
UPDATE community_levels SET prefix_connector = ' de '
WHERE name LIKE 'Level%' OR name LIKE 'Nível%' OR name LIKE '%Star%';
```

### 3. Na interface, montar o nome completo:

```typescript
// Função utilitária
function getCharacterDisplayName(character: Character, level: CommunityLevel): string {
  const prefix = level.display_prefix || level.name
  const connector = level.prefix_connector || ' '
  return `${prefix}${connector}${character.name}`
}

// Exemplos de resultado:
// "Fan de Freddie Mercury"         (Tier 1, Música)
// "Icon Freddie Mercury"           (Tier 5, Música)
// "Delta Wolverine"                (Tier 1, Marvel)
// "Omega Wolverine"                (Tier 5, Marvel)
// "Youngling Luke Skywalker"       (Tier 1, Star Wars)
// "Grand Master Luke Skywalker"    (Tier 5, Star Wars)
// "Genin Naruto Uzumaki"           (Tier 1, Naruto)
// "Hokage Naruto Uzumaki"          (Tier 5, Naruto)
// "Primeiro Ano de Harry Potter"   (Tier 1, Harry Potter)
// "Auror Harry Potter"             (Tier 4, Harry Potter)
```
