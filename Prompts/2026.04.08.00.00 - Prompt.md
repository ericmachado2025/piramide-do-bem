### ITEM 2 — Importação de IES (Ensino Superior) no Supabase

**Objetivo:** Popular a tabela `schools` com ~2.950 IES ativas do e-MEC, usando o
mesmo schema existente (name, city, state, school_type='superior').

**Fonte pública (acessível sem autenticação):**
```
https://raw.githubusercontent.com/nazareno/enade-vis/master/data/ies_Brasil.csv
```

**Formato do CSV:** separado por `;`, encoding UTF-8 BOM, colunas relevantes:
- `Instituição(IES)` → name
- `Município` → city
- `UF` → state
- `Organização Acadêmica` → (todos mapeiam para school_type = 'superior')
- `Situação` → filtrar apenas `'Ativa'`

**Script a criar:** `scripts/import_ies.py`

Lógica do script:
1. Baixar CSV da URL acima com requests ou urllib
2. Parsear com csv.DictReader(delimiter=';'), encoding='utf-8-sig'
3. Filtrar Situação == 'Ativa'
4. Deduplicar por (name_normalized, city, state) — CSV pode ter linhas repetidas
5. Normalizar nome: strip(). Se tudo maiúsculo, aplicar title().
6. Inserir em lotes de 100 via POST /rest/v1/schools com Prefer: resolution=ignore-duplicates
7. Usar service_role key no header Authorization e apikey
8. Imprimir total inserido vs ignorado ao final

A FURG já existe (name='Universidade Federal do Rio Grande (FURG)'). O
`resolution=ignore-duplicates` garante que não duplica.

Executar o script após criá-lo e reportar o resultado.

-- SELECT
DROP POLICY IF EXISTS "students_select_own" ON students;
CREATE POLICY "students_select_own" ON students

DROP POLICY IF EXISTS "students_update_own" ON students;
